# RHCSA Study Notes (RHEL 9)

This repository contains structured notes and practice exercises for preparing for the **Red Hat Certified System Administrator (RHCSA - EX200)** exam.  
The notes follow *RHCSA Red Hat Enterprise Linux 9: Training and Exam Preparation Guide (3rd Edition)* by Asghar Ghori, while mapping to the official **Red Hat exam objectives**.
